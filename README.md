# K-way-QuickSort
O algoritmo implementado é o QuickSort em K-partições, em todo e qualquer tipo de dado a ser utilizado.
Os integrantes do grupo são Igor Soares da Silva, Saimon Augusto da Silva Rocha, Adriel Borges Zanetti, Gabriel Anael Strotkamp e Jhônattan Djovane Reolon da Silva.
O QuickSort tem como objetivo ordernar um vetor partir de um pivô (sendo ele o primeiro elemento/o último elemento/um elemento aleatório do vetor), dividindo a tarefa de ordenação recursivamente.
